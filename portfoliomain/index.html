What I've Built
Featured Project
🔎
Phishing Website Detection System
Final Year Capstone Project · Machine Learning + Cybersecurity

Built a machine learning-powered system that classifies URLs and websites as phishing or legitimate. The system extracts structural, lexical, and content-based features from URLs and web pages, then uses trained classifiers to detect malicious intent in real time — helping protect users from credential theft, fraud, and identity attacks.

Python Scikit-learn Pandas Random Forest URL Feature Extraction Flask Machine Learning
Key Highlights
✓Extracted 20+ features including URL length, HTTPS presence, domain age, redirect count, and suspicious keywords
✓Trained and compared multiple classifiers — Random Forest, Decision Tree, and Logistic Regression — achieving high accuracy on benchmark datasets
✓Addresses a critical real-world threat: phishing accounts for over 36% of data breaches globally
✓End-to-end pipeline from raw URL input to classified output, with explainable feature importance
✓Practical application of both machine learning and cybersecurity domain knowledge
